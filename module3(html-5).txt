• How to embed audio and video in a webpage?

- The <vid eo> element allows us to embed video files into an HTML,
  very similar to the way images are embedded. Attributes we can include are:
  src This attribute stands for the source, which is very similar to the src
  attribute used in the image element.
  We will add the link to a video file in the src attribute.
